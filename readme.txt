AI-powered chatbot for educational institutions with multi-school support, white-label branding, and comprehensive application management.

== Description ==

EduBot Pro is a comprehensive AI-powered chatbot solution designed specifically for educational institutions. It streamlines the admissions process by providing intelligent, automated assistance to prospective students and parents.

= Key Features =

* **AI-Powered Conversations**: Advanced chatbot with natural language processing
* **Application Management**: Complete student application system with status tracking
* **Multi-School Support**: Manage multiple educational institutions from one dashboard
* **White-Label Branding**: Customize appearance to match your school's brand
* **Academic Configuration**: Flexible grade systems, academic years, and educational boards
* **Real-time Analytics**: Track conversations, applications, and conversion rates
* **Multi-channel Notifications**: Email, SMS, and WhatsApp integration
* **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices

= Supported Integrations =

* OpenAI GPT for intelligent responses
* WhatsApp Business API
* Email services (SMTP, Mailgun, SendGrid)
* SMS providers (Twilio, MessageBird)

= Use Cases =

* Student admission inquiries
* Fee structure information
* Course and curriculum details
* Campus tour scheduling
* Document requirements
* Application status tracking

= Academic Systems Supported =

* **Grade Systems**: K-12, Pre-K to 12, Nursery to 12, Class 1-10, Class 1-12
* **Academic Years**: Automatic current/next year calculation with support for April-March, June-May, September-August, January-December, and Custom periods
* **Educational Boards**: Multiple board configuration with CBSE, ICSE, State Boards, IB, IGCSE, and Custom options

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/edubot-pro` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the EduBot Pro menu in the WordPress admin to configure the plugin.
4. Configure your school information in the Settings section.
5. Set up your AI and notification integrations.
6. Add the chatbot to your website using the shortcode `[edubot_chatbot]` or `[edubot_application_form]`.

== Frequently Asked Questions ==

= Do I need an OpenAI API key? =

Yes, you need an OpenAI API key for the AI-powered responses. You can get one from https://platform.openai.com/

= Can I customize the chatbot appearance? =

Yes, the plugin includes comprehensive styling options and supports custom CSS for complete customization.

= Does it work with any WordPress theme? =

Yes, EduBot Pro is designed to work with any properly coded WordPress theme.

= Can I manage multiple schools? =

Yes, the plugin supports multi-school management with separate configurations for each institution.

= Is the plugin GDPR compliant? =

Yes, the plugin includes privacy features and allows you to configure data retention policies.

= Can I export application data? =

Yes, you can export applications in CSV format from the admin panel.

== Privacy Policy ==

EduBot Pro collects and processes the following data:
* Student and parent information submitted through application forms
* Chat conversation logs for improving service quality
* Analytics data for performance monitoring

All data is stored securely and can be deleted upon request. The plugin is GDPR compliant and includes data export/deletion features.
